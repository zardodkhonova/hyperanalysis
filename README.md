# HyperAnalysis

A Streamlit web application for predicting hypertension transition in patients using machine learning.

## Features
- Exploratory Data Analysis (EDA)
- Data Preprocessing with SMOTE
- LSTM and Random Forest models
- Time Series Predictions
- Personalized Health Recommendations

## Technologies
- Python, Streamlit, TensorFlow, Scikit-learn, Plotly, Pandas

## How to Run
```bash
pip install -r requirements.txt
streamlit run main.py
```
